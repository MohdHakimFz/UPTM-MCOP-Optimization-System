# UPTM-MCOP-Optimization-System
A comprehensive Java implementation of the Marketing Campaign Optimization Problem (MCOP) using multiple algorithmic approaches: Greedy, Dynamic Programming, Backtracking, and Divide &amp; Conquer. Includes advanced data structures (Min-Heap &amp; Splay Tree) with performance analysis. Developed for SWC3524/SWC4423 - Algorithmic Data Structures.
